# OJT Hours Counter - Project Tasks

## Project Overview
- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: MySQL

---

## Local Development Commands

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

---

## Project Structure

- `backend/` - Node.js Express API server
- `frontend/` - React + Vite frontend application

---

## Features

- User authentication (login/register)
- Daily OJT hours tracking
- Weekly report generation
- Excel export functionality
- Admin dashboard
